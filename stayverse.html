StayVerse

Designed and developed a full-stack hotel management system to manage room inventory, bookings, check-in/check-out, billing, and operational analytics. The system uses a Java-based backend with REST APIs and a web-based frontend to provide real-time availability, automated room allocation, booking history, invoice generation, and management dashboards.

Tech Stack

Backend: Java, JDBC, REST APIs Database: MySQL Frontend: HTML, CSS, JavaScript Architecture: Client–Server, MVC-style separation Tools: VS Code, MySQL Workbench

Implementation Screens

Key Features

  • Room availability tracking
  • Customer record management
  • Automated billing
  • Database-backed persistence

Upcoming Features

  • Visual floor-wise room layout with real-time availability indicators
  • Role-based access control for admin and staff users
